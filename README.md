## My personal DWM build

### Versions of DWM included in the more-dwm folder
	- dwm-3patches
	- dwm-4patches
	- dwm-5patches
	- dwm-3patches
	- dwm-customized-default
	- dwm-ultra-customized
	- dwm-vanitygaps-1patch

### My Personal DWM with Script in the /scripts folder contains:
	- scripts/batt
	- scripts/prayer
	- scripts/time
	- scripts/volume
	- scripts/fourtabs(turns off the bar for all tabs except tab 1)
	- scripts/weather
	- scripts/pywall

I basically modified most of DWM's original keybindings to better fit my own set of keybindings that I am usually used to.
